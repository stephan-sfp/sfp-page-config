# SFP Page Config

WordPress plugin for the School for Professionals network. Provides page configuration, course data dropdowns, sales page styling, sticky CTA, longread navigation, and brand theming across all five sites (DPS, DGA, CVD, DST, SFP).

## Installation

Download the latest release ZIP and install via WordPress > Plugins > Add New > Upload Plugin. The plugin includes a built-in auto-updater that checks this repository for new releases.
